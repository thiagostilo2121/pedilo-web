# 🤝 Contribuyendo a Pedilo

¡Gracias por tu interés en colaborar con Pedilo! Este es un proyecto abierto que busca democratizar la tecnología de pedidos para los negocios locales.

## 🚀 Guía de Inicio Rápido

1. **Fork** el repositorio.
2. Crea una rama para tu característica: `git checkout -b feature/nueva-funcionalidad`.
3. Realiza tus cambios y asegúrate de seguir los estándares del proyecto.
4. Ejecuta el linter y formateador:
   - `npm run lint`
5. **Push** a tu rama: `git push origin feature/nueva-funcionalidad`.
6. Abre un **Pull Request**.

## 🎨 Estándares de Código

### Frontend (React)
- **React 19** con hooks.
- **Tailwind CSS 4** para estilos.
- Componentes funcionales y atómicos.

## 🐛 Reportando Bugs
Si encuentras un error, por favor abre un **Issue** detallando:
- Pasos para reproducir.
- Comportamiento esperado vs comportamiento actual.
- Tu entorno (SO, navegador, versión de Node/Python).

## 🛡️ Seguridad
Para reportar vulnerabilidades, lee nuestro archivo [SECURITY.md](SECURITY.md).
